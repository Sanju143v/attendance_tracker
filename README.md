# Attendance Tracker - AttendEase

A professional attendance tracking system built with Flask.

## Features
- User & Admin authentication
- Check-in / Check-out with late detection
- Attendance history & monthly reports
- Leave request management
- User approval workflow

## Tech Stack
- **Backend:** Flask, SQLAlchemy, Flask-Login
- **Database:** SQLite
- **Frontend:** HTML, CSS (Dark Theme)

## Default Admin Login
- **Email:** admin@attendance.com
- **Password:** admin123

## Local Development
```bash
pip install -r requirements.txt
python run.py
```
